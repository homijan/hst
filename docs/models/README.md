# HST modeling

$`[ W_k f](x) = [\psi_k * f](x) = \int \psi_k(x^\prime) f(x - x^\prime) dx^\prime`$

$`R_0(z) = z + \exp^{i \arg (z)}`$

$`\rho(z) = i \ln (R_0(z))`$

$`S_0(x) = \rho(f(x))`$

$`S_m(x) = \rho([W_{k_m} S_{m-1}](x)) = \rho([\psi_k * S_{m-1}](x))`$
